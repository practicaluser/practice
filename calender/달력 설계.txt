달력 UI 형태 종류

달력은 단순히 “표 형태” 말고도 여러 UI 패턴이 있습니다.

1. 월간 그리드 (가장 일반적)
우리가 흔히 보는 달력
7열 × 5~6행
일정 관리에 적합
2. 주간 뷰 (Week View)
가로: 요일
세로: 시간 (예: 09:00 ~ 18:00)
일정 앱에서 많이 사용
3. 리스트형 (Agenda View)
날짜별로 세로 리스트
모바일에서 가독성 좋음
5월 1일
 - 일정 A
 - 일정 B

5월 2일
 - 일정 C
4. 카드형 캘린더
각 날짜를 카드처럼 표현
디자인 중심 UI
5. 히트맵형 (GitHub 스타일)
활동량 표시
색 농도로 데이터 표현
6. 타임라인형
시간 흐름 중심
프로젝트 일정 관리에 사용

<div> + CSS Grid 방식
장점
<table>보다 유연함 (반응형 대응 쉬움)
CSS로 레이아웃 제어가 직관적
요즘 프론트엔드에서 표준적인 접근

Flexbox 방식
display: flex + flex-wrap으로 7칸씩 줄바꿈
특징
Grid보다 제어는 조금 불편하지만 간단한 구현에는 충분
